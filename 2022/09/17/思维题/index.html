3.空瓶换酒问题

问题描述：

某酒店可以用a ( a > 0 ) 个空瓶换b ( 0 < b < a ) 瓶啤酒，假如一顾客一次性买了x ( x > 0 ) 瓶啤酒，问该顾客一共可以喝到多少瓶啤酒。

可以借空瓶：

​ 初始一共x个空瓶，每换一次空瓶减少 ( a - b ) 个，最后一次可以借空瓶，借完将换回来的酒的酒瓶还给对方。

ans=x+xa−b∗bans=x+\frac x {a-b}*b ans=x+a−bx​∗b

不可以借空瓶：

​ 可以先拿出a个空瓶，用借自己空瓶的形式将 ( x - a ) 个空瓶换成酒，最后剩余的 a 个空瓶还能再换一次。

ans=x+(x−aa−b+1)∗b=x+x−ba−b∗bans = x + ({ x-a \over a-b} +1 )*b\\ = x+{x-b \over a-b}*b ans=x+(a−bx−a​+1)∗b=x+a−bx−b​∗b

4.玻璃球

问题描述：

有一个100层的高楼，有两个玻璃球，问最少用几次一定能测试出玻璃球落下摔碎的最低楼层？

将100层楼平均分成 x 组，由低到高每组的最高层扔一次，若第k组玻璃球碎了，则答案在第k组楼层里，再在第k组楼层里从低到高扔第二颗玻璃球，碎的那一层即为答案。

ans=x+100x−1≥2x∗100x−1=19ans=x+{100 \over x}-1 \ge 2 \sqrt{ x*{100 \over x}}-1=19 ans=x+x100​−1≥2x∗x100​​−1=19

即最多19次就能测试出来
